# 実装ロードマップ (Implementation Roadmap)

## このドキュメントについて

- プロダクト全体の実装順序を**機能レベル**で定義する全体地図
- 各行は `/add-feature` の実行単位（詳細タスクは実行時に `.steering/` に生成される）
- 機能の実装が完了したら状態を `[x]` に更新し、ステアリングへのリンクを記載する
